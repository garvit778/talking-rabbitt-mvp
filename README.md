Talking Rabbitt 🐰

Talk to Your Data

Talking Rabbitt is a lightweight conversational analytics tool that allows users to upload a CSV file and ask questions about their data in natural language.

Instead of manually filtering Excel sheets or building dashboards, users can simply chat with their data and get instant answers along with automated visualizations.

The goal of this MVP is to demonstrate how Talking Rabbitt can replace a 10-minute manual Excel analysis with a 5-second conversation.

🚀 Features

Upload any CSV dataset

Ask natural language questions

AI converts the question into Pandas queries

Get clear text insights

Automatically generate charts and graphs

Lightweight and easy to use

Designed for non-technical users


A sample CSV file has been included in the repository to test the application.

You can use this file to:

Ask questions

Generate charts

Test conversational data queries
